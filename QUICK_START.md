## 🎯 最速デプロイ（3ステップ）

### ステップ1: GitHubにアップロード

```bash
# リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/product-qa-generator4.git
cd product-qa-generator4

# ZIPファイルを解凍してファイルを配置
unzip ~/Downloads/product-qa-premium-v4.7-complete.zip
mv product-qa-premium-v4.7/* .
mv product-qa-premium-v4.7/.gitignore .
rm -rf product-qa-premium-v4.7

# GitHubにプッシュ
git add -A
git commit -m "Deploy v4.7"
git push origin main
```

### ステップ2: Vercelにデプロイ

1. https://vercel.com にアクセス
2. 「Import Project」→ GitHubリポジトリを選択
3. 「Deploy」をクリック

### ステップ3: 使用開始

1. Vercel URLにアクセス（例: `https://your-project.vercel.app`）
2. OpenAI APIキーを入力
3. 商品URLを入力
4. 「Q&A生成」をクリック

**完了！** 🎉

## 💡 初回使用ガイド

### OpenAI APIキーの取得

1. https://platform.openai.com/api-keys にアクセス
2. 「Create new secret key」をクリック
3. キーをコピー（二度と表示されないので注意）
4. アプリに貼り付け

### 推奨設定

初めて使う場合は以下がおすすめ：

| 項目 | 推奨値 | 理由 |
|-----|-------|------|
| **Q&A数** | 指定しない（自動） | タイムアウト回避 |
| **リソースタイプ** | 両方 | 最も情報が豊富 |
| **プラン** | Vercel Hobby（無料） | 十分な性能 |

## 📖 基本的な使い方

### URL入力モード（推奨）

```
1. OpenAI APIキーを入力
2. 商品ページのURLを入力
   例: https://kutsu.com/products/example
3. リソースタイプを選択: 「両方」
4. Q&A数を選択: 「指定しない（自動）」
5. 「Q&A生成」をクリック
6. 待つ（30〜50秒）
7. PDFまたはRTFでダウンロード
```

### HTMLソースコードモード

アクセス制限があるページの場合：

```
1. ブラウザで商品ページを開く
2. 右クリック → 「ページのソースを表示」
3. HTMLをすべてコピー
4. アプリの「HTMLソースコード入力」タブをクリック
5. テキストエリアに貼り付け
6. 「Q&A生成」をクリック
```

## 🐛 よくある質問

### Q: Q&A生成ボタンをクリックしても反応しない

**A**: v4.7を使用していますか？v4.6にはバグがあります。

### Q: タイムアウトエラーが出る

**A**: 以下を試してください：
1. Q&A数を「指定しない（自動）」に変更
2. Q&A数を30問に減らす
3. リソースタイプを「指定URLのみ」に変更

### Q: HTMLソースコードタブが開けない

**A**: v4.7で修正済みです。最新版を使用してください。

### Q: APIキーエラーが出る

**A**: 以下を確認：
- キーが正しくコピーされているか
- OpenAIアカウントに課金設定があるか
- 使用制限に達していないか

### Q: 日本語が文字化けする

**A**: ダウンロード形式をRTFに変更してください。
